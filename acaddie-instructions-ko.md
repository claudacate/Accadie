# Acaddie — 사용 설명서
### 숏게임 어드바이저

---

## 빠른 참고

1. **거리 입력** — 핀까지의 야드 수를 입력합니다
2. **라이 선택** — 볼이 현재 놓인 위치를 선택합니다
3. **라이 세부 선택** — 서브메뉴가 나타나면 계속 선택합니다 (벙커 상태, 잔디 결, 수심)
4. **경사/스탠스 선택** — 발이 닿은 지면의 경사를 선택합니다
5. **장애물 선택** — 볼과 핀 사이에 장애물이 있으면 선택합니다 (기본값: 없음)
6. **「어드바이스 받기」탭** — 필수 항목이 모두 입력되면 버튼이 활성화됩니다
7. **샷 이름, 클럽, 번호별 지침**을 읽습니다
8. **경사 조정** 카드가 표시되면 반드시 확인합니다
9. **주의사항** 카드가 표시되면 반드시 확인합니다
10. **← 편집**으로 돌아가 수정하거나 **↺ 새 샷**으로 초기화합니다

---

## 전체 가이드

### 개요

Acaddie는 캐디처럼 작동하는 숏게임 어드바이저입니다. 거리, 라이, 경사, 장애물 등의 상황을 입력하면 어떤 샷을 쳐야 하는지, 어떤 클럽을 사용해야 하는지, 그리고 어떻게 치는지를 쉬운 말로 알려드립니다. 전문 용어 나열이 아닌, 실제로 도움이 되는 구체적인 지침을 제공합니다.

---

### 1단계 — 그린까지의 거리

거리 입력란을 탭하고 **핀**까지의 야드 수를 입력합니다 (그린 엣지까지가 아닙니다). 레이저 거리측정기나 GPS 워치에서 직접 읽은 수치를 사용하세요.

- 최솟값: 1야드
- 최댓값: 100야드
- 레이블 옆 초록 점이 켜지면 입력 완료를 의미합니다

> **팁:** Acaddie는 그린 주변의 숏게임 상황을 위해 설계되었습니다. 100야드를 초과하는 풀스윙 어프로치는 평소 클럽 거리 체계를 사용하세요.

---

### 2단계 — 라이

볼이 현재 놓인 상황을 가장 잘 설명하는 타일을 탭합니다. 6가지 선택지가 있습니다:

| 타일 | 사용 상황 |
|---|---|
| 🌿 페어웨이 / 프린지 | 짧은 잔디, 페어웨이, 또는 그린 바로 앞의 프린지 |
| 🌾 라이트 러프 | 러프에 있지만 볼의 대부분이 보이는 상태 |
| 🌳 딥 러프 | 볼이 묻혔거나 두꺼운 잔디에 대부분 가려진 상태 |
| 🏖️ 벙커 | 벙커 또는 모래 해저드 |
| 💧 얕은 물속 | 물속에 있지만 칠 수 있는 상태 (수심 2인치 미만) |
| 🪨 하드팬 / 맨땅 | 딱딱하고 건조한 지면, 카트 도로 가장자리, 또는 잔디 없는 맨땅 |

레이블 옆 초록 점이 켜지면 라이 선택 완료를 의미합니다.

---

### 3단계 — 라이 세부 정보 (상황별 서브메뉴)

특정 라이를 선택하면 추가 세부 정보를 묻는 서브메뉴가 자동으로 나타납니다. 어드바이스 엔진이 정확한 샷을 추천하기 위해 반드시 입력해야 합니다.

#### 벙커 세부 정보:
| 선택지 | 의미 |
|---|---|
| ✅ 클린 / 볼 위에 놓임 | 일반 벙커샷 — 볼이 모래 위에 놓여 있음 |
| 🔽 1/3 묻힘 | 볼이 모래에 약 3분의 1 정도 박혀 있음 |
| 🍳 프라이드에그 | 볼이 박혀 주변에 크레이터가 생긴 상태 (프라이드에그처럼 보임) |
| ❌ 완전히 묻힘 | 볼이 모래 표면 아래에 완전히 묻혀 있음 |
| 🔒 딥 플러그 | 볼이 깊이 박혀 있으며 벙커 벽 쪽에 끼어 있는 경우가 많음 |
| 🏌️ 엣지 근처 (칩샷) | 볼이 벙커 가장자리 근처에 있음 — 폭발샷 불필요, 칩샷으로 충분 |

#### 라이트 / 딥 러프 세부 정보:
| 선택지 | 의미 |
|---|---|
| ➡️ 결 방향 | 잔디가 그린 방향으로 자라고 있음 — 클럽이 잘 통과됨 |
| ⬅️ 역결 방향 | 잔디가 내 방향으로 자라고 있음 — 클럽이 걸리기 쉬움 |
| ⬇️ 수직 / 눌린 잔디 | 잔디가 아래로 눌려 있거나 볼이 둥지처럼 가라앉아 있는 상태 |

#### 얕은 물속 세부 정보:
| 선택지 | 의미 |
|---|---|
| 💧 1/3 잠김 | 볼이 3분의 1 물속에 — 상부 3분의 2가 보임 |
| 🌊 2/3 잠김 | 볼이 대부분 물속에 — 아주 작은 부분만 보임 |
| 🏊 완전 잠김 (2인치 미만) | 볼이 완전히 물속에 있지만 수심은 2인치 미만 |

> **참고:** 수심이 2인치를 초과하면 페널티 드롭을 선택하세요. Acaddie는 깊은 물속 샷은 권장하지 않습니다.

---

### 4단계 — 경사와 스탠스

**발이 닿은 지면**의 경사를 선택합니다 (그린의 경사가 아닙니다).

| 타일 | 의미 |
|---|---|
| ➖ 평지 | 지면이 평평함 — 일반 스탠스 |
| ⬆️ 오르막 | 목표 방향으로 올라가는 경사에서 치는 상황 |
| ⬇️ 내리막 | 목표 방향으로 내려가는 경사에서 치는 상황 |
| ↖️ 볼이 발보다 높음 | 볼이 발 위치보다 높은 사이드힐 라이 |
| ↘️ 볼이 발보다 낮음 | 볼이 발 위치보다 낮은 사이드힐 라이 |

레이블 옆 초록 점이 켜지면 경사 선택 완료를 의미합니다.

> **팁:** 오르막과 내리막이 헷갈릴 때는 어드레스 시 어깨가 자연스럽게 어느 쪽으로 기우는지 확인하세요. 오르막 = 오른쪽 어깨가 낮음 (오른손잡이 기준), 내리막 = 왼쪽 어깨가 낮음.

---

### 5단계 — 장애물

볼과 핀 사이에 장애물이 있으면 선택합니다. 기본값은 **없음** (초기 선택됨).

| 타일 | 의미 |
|---|---|
| ✅ 없음 | 통로가 열려 있음 — 모든 샷 유형 가능 |
| ☠️ 해저드 | 벙커, 연못, 또는 러프 지대가 있음 — 볼이 그 위를 넘어야 함 |
| 🌲 나무 / 낮은 장애물 | 나뭇가지, 펜스, 또는 낮은 장애물 — 볼을 낮게 날려야 함 |

> **중요:** 해저드 또는 나무를 선택하면 낮게 굴리는 샷은 제외됩니다. 장애물을 넘기에 충분한 높이와 궤도를 가진 샷만 추천됩니다.

---

### 6단계 — 어드바이스 받기

거리, 라이, 경사 세 가지 필수 항목의 초록 점이 모두 켜지면 **어드바이스 받기** 버튼이 활성화됩니다. 탭하면 어드바이스가 표시됩니다.

---

### 결과 화면 읽는 법

결과 화면은 최대 4개의 섹션으로 구성됩니다:

#### 상황 요약 바
화면 상단에 입력한 거리, 라이, 경사, 장애물을 한 줄로 요약합니다. 어드바이스를 읽기 전에 내용이 맞는지 확인하세요.

#### 추천 샷
- **샷 이름** — 큰 글자로 표시되는 기술 이름
- **클럽** — 일반 이름과 로프트 각도 모두 표시, 예: *샌드웨지 (SW · 56°)*. 두 가지 선택지가 표시되면 첫 번째가 우선 추천입니다.

#### 치는 방법
번호가 매겨진 단계별 지침을 쉬운 말로 설명합니다. 어드레스 포인트가 먼저, 스윙 포인트가 나중에 나옵니다. 샷을 치기 전에 모든 단계를 읽으세요.

#### ⛰ 경사 조정 *(평지 이외의 경사를 선택한 경우 표시)*
선택한 경사에 따른 스탠스, 에임 방향, 클럽 선택의 구체적인 조정 내용입니다. 반드시 메인 지침에 추가로 적용하세요.

#### ⚠ 주의사항 *(필요한 경우 표시)*
이 샷에서 예상되는 특별한 주의사항 — 비정상적인 볼 비행, 여분의 런닝, 고위험 기술, 또는 특별한 실행 포인트입니다. 샷을 결정하기 전에 반드시 읽으세요.

---

### 내비게이션

| 버튼 | 기능 |
|---|---|
| ← 편집 | 모든 선택을 유지한 채 입력 화면으로 돌아감 — 한 항목을 변경하고 새 어드바이스를 받을 수 있음 |
| ↺ 새 샷 | 모든 입력을 지우고 다음 샷을 위해 처음부터 시작 |

---

### 활용 팁

- **라이는 있는 그대로 입력하세요.** 클럽 선택에 가장 큰 영향을 미치는 것이 라이입니다. 확신이 없으면 더 나쁜 라이를 선택하세요 — 예상치 못한 상황에 놀라는 것보다 충분히 준비하는 편이 낫습니다.
- **러프의 잔디 결이 중요합니다.** 역결 러프는 순결보다 훨씬 치기 어렵습니다. Acaddie는 거리 손실을 고려하여 클럽 추천을 조정합니다.
- **벙커 라이 세부 정보는 매우 중요합니다.** 클린 폭발샷과 묻힌 라이는 완전히 다른 샷입니다. 탭하기 전에 2초를 들여 볼 상태를 확인하세요.
- **경사 조정은 메인 지침에 추가로 적용합니다.** 메인 지침이 기본 샷 방법을 알려주고, 경사 카드가 수정 방법을 알려줍니다. 둘 다 적용하세요.
- **장애물 선택은 사용 가능한 옵션을 제한합니다.** 통로가 열려 있는데 실수로 해저드를 선택하면 실제보다 보수적인 어드바이스가 표시될 수 있습니다. 결과 화면 상단의 상황 요약 바를 항상 확인하세요.
